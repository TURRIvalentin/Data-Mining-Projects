# Analysis of Teaching Methods using ANOVA

## Project Overview
A statistical evaluation of different teaching methods using Analysis of Variance (ANOVA). 

This project assesses whether the choice of teaching methodology has a statistically significant impact on student performance metrics, including hypothesis testing and post-hoc analyses.

## Tech Stack & Tools
- Language: R
- Environment: RStudio / RMarkdown
- Techniques: ANOVA, Hypothesis Testing, Statistical Inference

## How to Run the Project
1. Navigate to this specific project directory.
2. Open the .Rmd file in RStudio.
3. Knit the document to generate the HTML report.
