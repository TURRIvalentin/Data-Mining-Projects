# Correspondence and Network Analysis of Obesity-Related Data

## Project Overview
This project implements correspondence analysis and network graphing techniques to uncover hidden relationships within obesity-related datasets.

By analyzing categorical variables and their associations with different obesity levels, the analysis provides a visual and statistical representation of how lifestyle and demographic factors interact.

## Tech Stack & Tools
- Language: R
- Environment: RStudio / RMarkdown
- Techniques: Network Analysis, Correspondence Analysis, Graph Visualization

## How to Run the Project
1. Navigate to this specific project directory.
2. Open the .Rmd file in RStudio.
3. Knit the document to generate the HTML report.
